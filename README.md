## Install

1. Clone the repo: `$ git clone [URL]`

2. Install dependencies: `$ bundle`

3. Create the database `$ rails db:setup`

4. Run the app: `$ rails s`

5. Navigate to [http://localhost:3000/](http://localhost:3000/)

Use `$ rspec` to run tests
